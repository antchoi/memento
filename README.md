# sisyphus-hermes

Hermes-native Sisyphus plugin project.

This repository is being specified first through an Ouroboros Seed. The goal is to port the useful ideas of oh-my-openagent's Sisyphus/Ultraworker into a Hermes-native full plugin architecture: durable task lifecycle, Kanban-backed orchestration, profile-aware workers, review gates, and Telegram-friendly status reporting.

Current artifact:

- `.ouroboros/seeds/sisyphus-hermes.seed.yaml`

Implementation has not started yet.
